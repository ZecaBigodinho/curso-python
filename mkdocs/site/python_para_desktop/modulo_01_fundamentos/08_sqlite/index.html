    SQLITE¶

    O que é SQLite?¶

    SQLite é um SGBD relacional leve, que armazena o banco inteiro em um único arquivo. Não requer servidor separado — a biblioteca é embutida na aplicação. Muito usado em aplicativos mobile, desktop e prototipação.

    Vantagens¶

    • Zero configuração.
    • Portabilidade total (arquivo único).
    • Suporte completo a SQL.
    • Integração nativa com Python via módulo sqlite3.
    • Ideal para projetos pequenos e médios.

    Exemplo via Terminal¶

    sqlite3 meu_banco.db
CREATE TABLE contatos (id INTEGER PRIMARY KEY, nome TEXT);
INSERT INTO contatos VALUES (1, 'João');
SELECT * FROM contatos;
.quit

    Curiosidade: O SQLite é o banco de dados mais usado no mundo! Está presente em todos os smartphones Android/iOS, navegadores Chrome/Firefox, sistemas embarcados e até na aviação.

    PYTHON + SQLITE (CRUD COMPLETO)¶

    Conexão e Criação de Tabela¶

    import sqlite3

# Conecta (cria arquivo se não existir)
conn = sqlite3.connect("cadastro.db")
cursor = conn.cursor()

# Cria tabela
cursor.execute("""
    CREATE TABLE IF NOT EXISTS pessoas (
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        nome TEXT NOT NULL,
        email TEXT UNIQUE
    )
""")
conn.commit()

    Create (Inserir)¶

    def inserir(nome, email):
    cursor.execute("INSERT INTO pessoas (nome, email) VALUES (?, ?)", (nome, email))
    conn.commit()
    print("Registro inserido!")

inserir("Maria", "maria@email.com")

    Read (Consultar)¶

    def listar():
    cursor.execute("SELECT * FROM pessoas")
    for linha in cursor.fetchall():
        print(linha)

listar()

    Update (Atualizar)¶

    def atualizar(id, novo_nome):
    cursor.execute("UPDATE pessoas SET nome = ? WHERE id = ?", (novo_nome, id))
    conn.commit()

atualizar(1, "Maria Silva")

    Delete (Excluir)¶

    def excluir(id):
    cursor.execute("DELETE FROM pessoas WHERE id = ?", (id,))
    conn.commit()

excluir(1)

    Fechando Conexão¶

    conn.close()

    Boa prática: Sempre use placeholders ? para evitar SQL Injection. Nunca concatene strings diretamente na query.
