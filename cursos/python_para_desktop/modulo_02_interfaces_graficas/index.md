# Módulo 02: Interfaces Gráficas com Python

## Visão Geral
Neste módulo, você aprenderá a sair da linha de comando (CLI) e construir aplicações com janelas, botões e interatividade visual usando as bibliotecas Tkinter e CustomTkinter.

## Capítulos Planejados
1. Introdução ao Tkinter
2. Gerenciamento de Layout
3. Eventos e Interatividade
4. CustomTkinter e Interfaces Modernas
5. Projeto Prático do Módulo 02
