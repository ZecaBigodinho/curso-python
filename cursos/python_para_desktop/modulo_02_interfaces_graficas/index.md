# Módulo 02 — Interfaces Gráficas com Python

> **Curso:** python_para_desktop  
> **Módulo:** 2 de 3  
> **Descrição:** Neste módulo, você aprenderá a sair da linha de comando (CLI) e construir aplicações com janelas, botões e interatividade visual usando as bibliotecas Tkinter e CustomTkinter.

---

## 📋 Visão Geral do Módulo

O desenvolvimento desktop requer uma interface gráfica de usuário (GUI) para garantir usabilidade. Este módulo cobre desde a estrutura básica de janelas até a construção de aplicações modernas com bordas arredondadas, modo escuro e gráficos interativos.

## 🎯 Objetivos do Módulo

Ao final deste módulo, você será capaz de:

- [x] Construir janelas interativas com componentes padrão (Tkinter) e modernos (CustomTkinter).
- [x] Posicionar elementos na tela usando gerenciadores de layout dinâmicos.
- [x] Capturar e tratar eventos (cliques de mouse, pressionamento de teclas) e interagir com o usuário de forma bidirecional.
- [x] Estruturar um projeto real de dashboard utilizando múltiplos frames, gráficos e armazenamento local de dados (JSON).
