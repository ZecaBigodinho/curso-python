# Módulo 03 — Banco de Dados e Configurações

> **Curso:** python_para_desktop  
> **Módulo:** 3 de 3  
> **Descrição:** Neste módulo final, você aprenderá a conectar suas interfaces a um banco de dados relacional (SQLite), gerenciar senhas com `.env`, organizar o código na arquitetura MVC e empacotar tudo num executável.

---

## 📋 Visão Geral do Módulo

Para que uma aplicação desktop seja profissional, não basta ser bonita. Ela precisa salvar dados permanentemente (persistência) e proteger chaves e configurações. Você irá consolidar todo o aprendizado em um projeto final empacotável.

## 🎯 Objetivos do Módulo

Ao final deste módulo, você será capaz de:

- [x] Criar tabelas, inserir e consultar dados em bancos SQLite.
- [x] Conectar elementos gráficos (como Treeview e campos de formulário) ao banco de dados com segurança (contra SQL Injection).
- [x] Proteger credenciais e chaves usando variáveis de ambiente (`.env`).
- [x] Estruturar aplicações completas utilizando o padrão MVC (Model-View-Controller).
- [x] Empacotar sua aplicação com o PyInstaller para distribuí-la como executável sem depender de instalações complexas.
