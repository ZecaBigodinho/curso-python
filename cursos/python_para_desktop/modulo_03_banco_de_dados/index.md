# Módulo 03: Bancos de Dados e Configurações

## Visão Geral
Neste módulo, você aprenderá a criar aplicações robustas conectadas a um banco de dados relacional (SQLite/MySQL), como gerenciar configurações e senhas com arquivos `.env`, e organizar a arquitetura do seu projeto para o App Python Final.

## Capítulos Planejados
1. Introdução a Banco de Dados e SQLite
2. Operações CRUD em Python
3. Variáveis de Ambiente (.env) e Segurança
4. Juntando tudo (Arquitetura de Software)
5. Projeto Final do Curso (App Completo)
