## Questão 2

O que acontece se você omitir o parâmetro Z na função abaixo?

```python
def calcular(a, b, Z=None):
    pass
```

- [ ] A) Erro de sintaxe
- [ ] B) O valor assume None
- [ ] C) A aplicação trava

??? success "Resposta e Explicação"
    **Resposta correta:** B) O valor assume None
    
    Como o parâmetro possui um valor padrão definido na assinatura, a falta dele na chamada faz o interpretador assumir `None`.
